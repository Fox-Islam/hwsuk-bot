# A Discord bot made for HardwareSwapUK community

This bot is a smaller version of the main [Discord Bot](https://github.com/sachinlim/discord-bot) project. It incorporates the `!search` command and has renamed it to `!ebay` because there are other website specific commands. The bot will also shorten links from eBay UK and Amazon UK.

### HardareSwapUK

HardwareSwapUK is a [community on Reddit](https://www.reddit.com/r/HardwareSwapUK/) where people can buy, sell or trade hardware in the UK. It also exists on Discord but the origin of the community is on Reddit. The bot is implemented on the Discord server.
